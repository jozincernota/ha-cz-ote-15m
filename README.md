\# CZ OTE 15min Spot (ha-cz-ote-15m UI)



Plně UI Home Assistant integrace s 15minutovými spot cenami OTE (CZ):



\## Funkce



\- 96 hodnot / den (15min intervaly)

\- Aktuální cena

\- Min / Max / Průměr dne

\- Nejlevnější 2h blok

\- Přepočet jednotek CZK/kWh

\- Volba zobrazování zítřejších cen

\- Plně UI (Config Flow)

\- Připravené pro grafy (ApexCharts, History Graph)



\## Instalace (HACS)



1\. Přidej repozitář: `https://github.com/jozincernota/ha-cz-ote-15m`

2\. Kategorie: \*\*Integration\*\*

3\. Install

4\. Restart Home Assistant

5\. Settings → Devices \& Services → Add Integration → \*\*CZ OTE 15min Spot\*\*

6\. Nastav jednotky a volby přes UI
